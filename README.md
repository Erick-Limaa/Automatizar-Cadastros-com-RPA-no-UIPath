# Automatizar-Cadastros-com-RPA-no-UIPath
Esta automação foi desenvolvida como parte do processo seletivo da Redesign Consultoria.
O objetivo é extrair os dados do relatório de oportunidades de vendas e realizar o preenchimento automático do formulário correspondente.
